## Fake news site

https://fakenewsgetr.herokuapp.com/

# Installation

    pip install virtualenv
    virtualenv venv
    source venv/bin/activate
    pip install -r requirements.txt

# Fetching fake news

    cd YOURPROJECTDIRECTORY
    source venv/bin/activate
    python fakenews.py fetch

# Push to Heroku

    git push heroku master
